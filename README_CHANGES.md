# smdownbot — Пользовательские настройки (changelog)

Добавлено персональное меню настроек для каждого пользователя: **смена языка,
хештеги источника, канал источника, режим аудио/видео**.

Открывается кнопкой `⚙️ Настройки` в главном меню или командой `/settings`
(только в личном чате).

## Что изменилось по файлам

### `database/models.py`
- Миграция: в таблицу `users` добавлены 3 колонки (безопасно для существующей БД):
  - `show_tags INTEGER DEFAULT 0` — показывать хештеги источника в подписи
  - `show_source_channel INTEGER DEFAULT 0` — показывать канал/@логин источника
  - `download_mode TEXT DEFAULT 'ask'` — режим по умолчанию: `ask` / `video` / `audio`
- В `DEFAULT_SETTINGS` добавлен `feature_user_settings = "1"` — общий рубильник
  меню настроек (админ может выключить из своей панели).

### `database/db.py`
Новые функции:
- `get_user_settings(user_id)` → dict с дефолтами
- `toggle_user_flag(user_id, field)` — переключение флага (только из белого списка
  `show_tags` / `show_source_channel`, защита от SQL-инъекций)
- `get_user_download_mode(user_id)` / `set_user_download_mode(...)` — с валидацией

### `keyboards/inline.py`
- `settings_keyboard()` — панель из 5 строк с текущими значениями
- `settings_language_keyboard()` — выбор языка с кнопкой «Назад»
- `settings_mode_keyboard()` — выбор режима (✓ у текущего)
- `main_menu_keyboard(..., show_settings=True)` — добавлена кнопка `⚙️ Настройки`

### `services/i18n.py`
- Новые ключи для **en / ru / uk**: `btn_settings`, `cmd_settings`,
  `settings_title`, `set_opt_*`, `opt_on/off`, `mode_*`, `choose_mode`,
  `btn_back`, `btn_close`, `setting_saved`.

### `handlers/settings.py` (НОВЫЙ)
- Роутер: команда `/settings`, кнопка `⚙️ Настройки`, и все колбэки `uset:*`
  (переключение флагов, смена языка, выбор режима, назад/закрыть).
- Уважает рубильник `feature_user_settings`; работает только в личных чатах.

### `handlers/start.py`
- `_menu_flags()` теперь возвращает и `show_settings`; кнопка появляется в меню,
  если функция включена.

### `handlers/download.py`
- **Режим загрузки:** в личке используется персональный `download_mode`
  пользователя; в группах — как раньше, админский `group_download_mode`.
  `video` → скачать видео сразу, `audio` → аудио сразу, `ask` → показать кнопки.
- **Подпись:** хештеги и канал источника теперь считаются как
  `админский_глобальный ИЛИ персональный_пользователя` — то есть глобальный
  флаг админа = «принудительно для всех», а по умолчанию каждый решает сам.

### `handlers/admin.py` + `services/i18n_admin.py`
- В админ-панель добавлен переключатель `feature_user_settings`
  (`⚙️ Меню настроек юзера`) — можно выключить всё меню разом.

### `bot.py`
- Зарегистрирован `settings.router` (после `start`, до `download`).
- `/settings` добавлена в публичное меню команд.

## Установка
Распакуйте архив поверх проекта (структура папок сохранена). Зависимостей
новых нет. При первом запуске миграция добавит колонки автоматически —
существующие данные не теряются.
